基本觀念

程式利用 Stream 物件 把放在Source的資料讀取出來

程式亦可以利用 Stream 物件 把資料寫到資料來源(source)中

基本上, 要使用io 動作,則要 import java.io package.

第一部分. 文字方面的串流 (Character Stream)

  • Reader abstract super class: for reading 16-bit characters

    所有的文字讀取串流 ==> 全部都繼承自 Reader

                       範例:

 InputStreamReader is = new InputStreamReader(System.in);   

 char x[]=new int[1];

 is.read(x);

    

　

  • Writer abstract super class: for writing 16-bit characters

    所有的文字寫入串流 ==> 全部都繼承自 Writer

                    範例:

 OutputStreamWriter os= new OutputStreamWriter(System.out);

 os.write("Hello");

 os.close();                // 將資料寫出去

　

第二部分 二元檔案串流 (byte stream)

所有的 Byte 寫入串流 ==> 全部都繼承自 InputStream

  • InputStream: for reading 8-bit bytes

　

  • OutputStream: for writing 8-bit bytes

效率的問題

=> 可以先把資料放到buffer中,接著一行一行的讀出來.

=> 如何做? (利用 BufferedReader 把 FileReader 包起來)

1. FileReader FileStream=new FileReader("c:\\test\\test.txt");

2. BufferedReader BufferedStream=new BufferReader(FileStream);

3. BufferedStream.readLine(); // 傳回一行

注意: 當檔案結束時, readLine() 會傳回 null值.

import java.io.*;
public class ReadFile2 {
    public static void main(String arg[]) throws Exception
    {
        // 1. 建立文字檔讀取物件
        FileReader FileStream=new FileReader("c:\\test\\test.txt"); 
        
        // 2. 透過 BufferedReader 幫我們讀資料
        BufferedReader BufferedStream=new BufferedReader(FileStream);
	 
         //讀取資料
     String data;
     do{
         // 3. 接著 一行一行 的把資料從檔案中讀出來
            data=BufferedStream.readLine();
         // 4. 當讀取到最後一行後,
            if(data==null)
                break;          // 讀到檔案結束
           System.out.println(data);
        } while(true);      
    } // end of main
}

範例: CopyFile.java

將 line.txt 本文檔copy 到 linenum.txt, 拷貝的過程中順便將行號加上去.

1. 開兩個 stream, 一個是 Read 另一個是Write.

2. 利用 ready() 判斷是否讀到檔尾

3. 如何把數字加進檔案中?

  • 數字 --> 字串: String.valueOf(數字);

  • write (數字字串)

練習一下:

建立一個檔案, 內容為 1 2 3 4 5 .... 100

4. 完整程式

CopyFile.java

/********************* CopyFile.java ***********************/
import java.io.*;
public class CopyFile
  {
  public static void main(String args[]) throws Exception
    {
    BufferedReader br = new BufferedReader(
                                                                   new FileReader("line.txt"));
    BufferedWriter bw = new BufferedWriter(
                                                                   new FileWriter("linenum.txt"));
    String space="  ";
    int num=0;
   // 判斷是否可以讀檔
    while (br.ready()){
         // 由line.txt 中讀取一行字串		
          String s=br.readLine();	
         // 若讀到檔尾時,則break 
          if (s==null) 
                   break;                                
          num++;
          bw.write(String.valueOf(num));     // 把數字寫到linenum.txt中
          bw.write(space);                           
          bw.write(s);                                   // 寫入剛讀取的字串
          bw.newLine();                               // 寫入換行
     }
    bw.close();
    }
  }

進階部分

範例: Token.java

利用 Token的方式,將讀進來的資料進行剖析,分成

1. 數字 2. 文字

進行處理.

=>

1. 建立一個 BufferedRead 的stream=> 讀檔的工作

2.利用 StreamTokenizer 物件幫我們處理辨別的工作

例: (把 input stream丟給 StreamTokenizer處理)

StreamTokenizer st= new StreamTokenizer(讀取檔案Stream);

設定 StreamTokenizer

1.把不要處理的東西設定成空白:

st.whitespaceChars(0,32); // 指定 0<= c <=32的character全設為空白

2.是否要判斷 end of line

如: st.eollsSignificant(true); // 開啟判斷 end of line的功能

OK, 設定好了,現在如何作 ?

利用 int TokenType= st.nextToken() 得到目前的字串是屬於哪一類?

1. st.TT_NUMBER: 數字

2. st.TT_WORD: 文字

3. st.TT_EOF: end of file

4. st.TT_EOL: end of line

3. 完整程式範例 Token.java

Token.java

/*************************** Token.java ******************/
import java.io.*;
public class Token
  {
  public static void main(String args[]) throws IOException
    {
    BufferedReader br = new BufferedReader(
                        new FileReader("linenum.txt"));
    BufferedWriter bw = new BufferedWriter(
                        new FileWriter("token.txt"));
    StreamTokenizer st=new StreamTokenizer(br);
    int tokenType;
    st.whitespaceChars(0,32);               //0-32為空白字元
    st.eolIsSignificant(true);
    while (br.ready())
      {
      tokenType=st.nextToken();
      switch (tokenType)
        {
        case StreamTokenizer.TT_NUMBER:
          bw.write(String.valueOf((long)st.nval));// 把分析出來的數字寫到檔案中
          bw.newLine();
          break;
        case StreamTokenizer.TT_WORD:
          bw.write(st.sval);                               // 把Word 寫到檔案中
          bw.newLine();
          break;
        default:
          ;
        }
      }
    bw.close();
    }
  }
